## Классы и объекты

### Класс

**Класс** — это пользовательский тип данных, который описывает структуру и поведение объектов.

Класс можно представить как чертёж автомобиля.

Он описывает:
- какие данные будут храниться;
- какие действия можно выполнять.

```csharp
class Car
{
}
```

---

### Объект

**Объект** — это экземпляр класса, созданный в памяти.

```csharp
Car ferrari = new Car();
```

Здесь:

```text
Car      → класс (тип)
ferrari  → объект класса Car
```

---

## Создание объекта

```csharp
Car ferrari = new Car();
```

Оператор `new`:

1. Выделяет память под объект.
2. Вызывает конструктор.
3. Возвращает ссылку на созданный объект.

---

## Примеры объектов

```csharp
Car ferrari = new Car();
Random rnd = new Random();
```

Также массив является объектом:

```csharp
int[] numbers = new int[2];
```

Потому что массивы в C# — ссылочные типы.

---

## Ссылки на объекты

```csharp
Car ferrari = new Car();

Car lamborghini = ferrari;
```

Здесь новый объект **не создаётся**.

Копируется только ссылка.

---

### Что происходит в памяти?

После:

```csharp
Car ferrari = new Car();
```

```text
ferrari ───► Car object
```

После:

```csharp
Car lamborghini = ferrari;
```

```text
ferrari ──────┐
              ▼
         Car object
              ▲
lamborghini ──┘
```

Обе переменные указывают на один и тот же объект.

---

## Пример

```csharp
Car ferrari = new Car();
Car lamborghini = ferrari;

Console.WriteLine(
    Object.ReferenceEquals(ferrari, lamborghini));
```

Результат:

```text
True
```

Потому что ссылка одна и та же.

---

## Кратко

```text
Класс  → описание объекта (чертёж)

Объект → экземпляр класса в памяти

new    → создаёт объект

Переменная класса хранит ссылку на объект
```
