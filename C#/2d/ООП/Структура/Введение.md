## Структуры (`struct`)

**Структура** — это пользовательский тип значения.

По назначению похожа на класс, но имеет другое поведение в памяти.

---

## Главное отличие от класса

### Класс

```csharp
class Player
{
    public int X;
}
```

Класс — ссылочный тип.

При присваивании копируется ссылка:

```csharp
Player p1 = new Player();
Player p2 = p1;
```

Обе переменные указывают на один объект.

---

### Структура

```csharp
struct Vector2
{
    public int X;
    public int Y;
}
```

Структура — тип значения.

При присваивании копируются данные:

```csharp
Vector2 a = new Vector2();
Vector2 b = a;
```

Получаются две независимые копии.

---

## Объявление структуры

```csharp
struct Vector2
{
    public int X;
    public int Y;
}
```

Использование:

```csharp
Vector2 pos = new Vector2();

pos.X = 10;
pos.Y = 20;
```

---

## Конструктор структуры

Можно создавать собственные конструкторы:

```csharp
struct Vector2
{
    public int X;
    public int Y;

    public Vector2(int x, int y)
    {
        X = x;
        Y = y;
    }
}
```

Использование:

```csharp
Vector2 pos = new Vector2(10, 20);
```

---

## Важное правило

В конструкторе структуры необходимо инициализировать все поля:

```csharp
public Vector2(int x, int y)
{
    X = x;
    Y = y;
}
```

Нельзя:

```csharp
public Vector2(int x)
{
    X = x;
}
```

Потому что поле `Y` останется неинициализированным.

---

## Использование `this()`

Поэтому пишут:

```csharp
public Vector2(int x) : this()
{
    X = x;
}
```

`this()` сначала заполняет все поля значениями по умолчанию:

```csharp
X = 0;
Y = 0;
```

После чего можно изменить нужные поля.

---

## Копирование структуры

```csharp
Vector2 a = new Vector2(10, 20);
Vector2 b = a;

b.X = 100;
```

Результат:

```csharp
Console.WriteLine(a.X); // 10
Console.WriteLine(b.X); // 100
```

Изменение `b` не влияет на `a`.

---

## Когда использовать структуры?

Обычно для небольших объектов, которые представляют одно значение:

```text
Координаты
Размеры
Цвета
Точки
Векторы
Дата и время
```

Примеры из .NET:

```csharp
DateTime
TimeSpan
Guid
Point
Vector2
```

---

## Когда лучше использовать класс?

Если объект:

- большой;
- часто изменяется;
- должен передаваться по ссылке;
- имеет сложное поведение.

Тогда обычно используют `class`.

---

## Кратко

```text
class  → ссылочный тип
struct → тип значения
```

```csharp
Vector2 a = new Vector2(10, 20);
Vector2 b = a;
```

Для структуры:

```text
a и b — разные копии данных
```

Для класса:

```text
a и b — ссылки на один объект
```

Структуры удобно использовать для небольших объектов вроде координат, размеров и векторов.
