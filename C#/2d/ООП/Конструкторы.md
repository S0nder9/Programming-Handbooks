## Конструкторы

**Конструктор** — это специальный метод класса, который вызывается автоматически при создании объекта.

Он используется для:
- инициализации полей;
- задания значений по умолчанию;
- проверки входных данных.

---

## Пример класса

```csharp
class Car
{
    public string Name;
    private int _age;
    public int Years;
```

---

## Конструктор с параметрами

```csharp
public Car(string name, int years)
{
    if (years < 0)
    {
        Years = 0;
    }
    else
    {
        Name = name;
        Years = years;
    }
}
```

### Что происходит:

```text
При создании объекта:
→ передаём name и years
→ выполняется проверка
→ поля инициализируются
```

---

## Конструктор по умолчанию

```csharp
public Car()
{
    Name = "K";
    Years = 10;
}
```

Он вызывается, если параметры не переданы:

```csharp
Car car = new Car();
```

---

## Перегрузка конструктора

В классе может быть несколько конструкторов:

```csharp
Car(string name, int years)
Car()
```

Это называется **перегрузка конструктора**.

---

## Использование объекта

```csharp
Car fer = new Car("F40", 10);
fer.ShowTP();
```
