## ООП (Объектно-ориентированное программирование)

Основные понятия ООП:

- **Класс** — шаблон (описание объекта).
- **Объект** — экземпляр класса, созданный по этому шаблону.

Например:

```csharp
Customer customer = new Customer();
```

`Customer` — класс, `customer` — объект.

---

## 1. Абстракция

Выделение только важных характеристик объекта и скрытие лишних деталей.

Пример:

```csharp
Car.StartEngine();
```

Для запуска двигателя не нужно знать, как работают поршни и свечи зажигания.

---

## 2. Инкапсуляция

Сокрытие внутренних данных объекта и предоставление доступа к ним через свойства и методы.

```csharp
private string _name;

public string Name
{
    get => _name;
    set => _name = value;
}
```

Пользователь класса работает с `Name`, а не напрямую с `_name`.

---

## 3. Наследование

Позволяет создавать новый класс на основе существующего.

```csharp
class Animal
{
    public void Eat() { }
}

class Dog : Animal
{
}
```

`Dog` автоматически получает метод `Eat()` от `Animal`.

---

## 4. Полиморфизм

Позволяет использовать один интерфейс для разных объектов.

```csharp
Animal animal = new Dog();
animal.MakeSound();
```

Один и тот же вызов метода может выполнять разное поведение в зависимости от типа объекта.

---

## Кратко

```text
Абстракция   → показываем главное, скрываем детали
Инкапсуляция → защищаем данные внутри класса
Наследование → создаём класс на основе другого
Полиморфизм  → один интерфейс, разное поведение
```
