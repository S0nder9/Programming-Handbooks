## Методы

Метод — это функция, которая принадлежит классу и описывает поведение объекта.

Он позволяет объекту **выполнять действия** и изменять своё состояние.

---

## Пример класса `Car`

```csharp
class Car
{
    public string Name = "1";
    private string _age = "0";
    public int Years = 0;
```

---

## Метод вывода информации

```csharp
public void ShowTP()
{
    Console.WriteLine($"Название авто: {Name}, {_age}");
}
```

### Что делает:
- выводит состояние объекта;
- показывает текущие значения полей.

---

## Метод изменения состояния

```csharp
public void BecomeOlder(int Years, int runAway)
{
    _age += Years + this.Years;
}
```

---

## Важно про `this`

```csharp
this.Years
```

`this` — это ссылка на **текущий объект класса**.

### Зачем он нужен:

Когда имя параметра совпадает с полем:

```csharp
public int Years;
public void BecomeOlder(int Years)
```

то:

```text
Years (без this) → параметр метода
this.Years       → поле объекта
```

---

## Пример различия

```csharp
this.Years = 10;  // поле класса
Years = 10;       // параметр метода
```

---

## Использование в `Main`

```csharp
Car fer = new Car();

fer.ShowTP();
fer.BecomeOlder(5, 10);
fer.ShowTP();
```

---

## Что происходит

1. Создаётся объект `Car`
2. Вызывается `ShowTP()` → вывод состояния
3. Вызывается `BecomeOlder(5, 10)` → изменение состояния
4. Снова `ShowTP()` → уже новые данные

---

## Кратко

```text
Метод = поведение объекта

ShowTP()      → показывает состояние
BecomeOlder() → изменяет состояние

this → ссылка на текущий объект
```
