## Инкремент и декремент

**Инкремент (`++`)** — увеличивает значение на 1.  
**Декремент (`--`)** — уменьшает значение на 1.

---

```csharp
int i = 0;

i++; // i = 1
i--; // i = 0

++i; // i = 1
--i; // i = 0
```

---

## Постфиксный (`i++`, `i--`)

Сначала используется текущее значение, **потом** изменяется.

```csharp
int x = 5;

Console.WriteLine(x++); // 5
Console.WriteLine(x);   // 6
```

---

## Префиксный (`++i`, `--i`)

Сначала изменяется значение, **потом** используется.

```csharp
int x = 5;

Console.WriteLine(++x); // 6
Console.WriteLine(x);   // 6
```

---

## Приоритет

У `++` и `--` очень высокий приоритет, поэтому они выполняются раньше многих операций.

---

## Разбор примера

```csharp
int a = 0;

Console.WriteLine((++a) + 2 + 1 + (a++) + "1" + (++a) * 2);
```

Шаги:

1. `++a` → `a = 1`, возвращает `1`
2. `+ 2`
3. `+ 1`
4. `a++` → возвращает `1`, потом `a = 2`
5. `"1"` → начинается **конкатенация строки**
6. `++a` → `a = 3`
7. `3 * 2 = 6`

Получаем:

```text
1 + 2 + 1 + 1 = 5
```

Дальше:

```text
"5" + "1" + "6"
```

Результат:

```text
516
```
