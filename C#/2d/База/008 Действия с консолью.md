## Консоль

```csharp
// Решение проблемы с языком
Console.InputEncoding = Encoding.Unicode;
Console.OutputEncoding = Encoding.Unicode;

string name;
name = Console.ReadLine(); // считывает строку, которую мы вводим в консоль и сохраняет ее в переменной name

Console.WriteLine(name); // вывод в консоль
Console.Write(name); // вывод в консоль без новой строки


int age = 27;
age = Convert.ToInt32(Console.ReadLine()); // Конвертация в int, так как ReadLine - строка


Console.ReadKey(); // консоль ждет от нас нажатия клавиши, чтобы не закрыться сразу после выполнения 
```
