## Двумерные массивы

**Двумерный массив** — это массив, который имеет **2 измерения**: **строки** и **столбцы**.

Его можно представить как **таблицу**:

```text
[1] [2] [3]
[4] [5] [6]
[7] [8] [9]
```

Каждый элемент определяется двумя индексами:

```csharp
array[строка, столбец]
```

Например:
```csharp
array[0, 1]
```

означает:
- строка `0` (первая строка),
- столбец `1` (второй столбец).

---

## Создание двумерного массива

```csharp
int[,] array2D = new int[3, 3];
```

Создаётся массив 3×3:

```text
[0] [0] [0]
[0] [0] [0]
[0] [0] [0]
```

По умолчанию `int` заполняется нулями.

---

## Инициализация значениями

```csharp
int[,] array2DIn =
{
    { 1, 2 },
    { 3, 4 }
};

// или

 int[,] array4 = new int[2, 3] { { 2, 3, 4 }, { 1, 3, 4 } };
```

Получаем:

```text
[1] [2]
[3] [4]
```

---

## Доступ к элементу

```csharp
Console.WriteLine(array2DIn[0, 0]);
```

Результат:
```text
1
```

---

```csharp
Console.WriteLine(array2DIn[1, 0]);
```

Результат:
```text
3
```

---

## Изменение элемента

```csharp
array2DIn[0, 0] = 5;
Console.WriteLine(array2DIn[0, 0]);
```

Результат:
```text
5
```

Теперь массив выглядит так:

```text
[5] [2]
[3] [4]
```

---

## Размер массива

Количество строк:

```csharp
array2DIn.GetLength(0)
```

Количество столбцов:

```csharp
array2DIn.GetLength(1)
```

Пример:

```csharp
Console.WriteLine(array2DIn.GetLength(0)); // 2
Console.WriteLine(array2DIn.GetLength(1)); // 2
```

---

## Перебор двумерного массива

Используются вложенные циклы:

```csharp
for (int i = 0; i < array2DIn.GetLength(0); i++)
{
    for (int j = 0; j < array2DIn.GetLength(1); j++)
    {
        Console.Write(array2DIn[i, j] + " ");
    }
    Console.WriteLine();
}
```

Вывод:
```text
5 2
3 4
```

## Пример: поле "Крестики-нолики"

```csharp
string[,] ticTacToe =
{
    { "O", "X", "X" },
    { "O", "O", "X" },
    { "X", "X", "O" }
};
```

Получить элемент:

```csharp
Console.WriteLine(ticTacToe[1, 2]);
```

Результат:
```text
X
```

(2 строка, 3 столбец)
