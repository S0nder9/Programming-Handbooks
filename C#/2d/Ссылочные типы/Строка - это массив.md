## Строка — это массив?

Не совсем.

В C# `string` — это **класс (ссылочный тип)**, а не массив.

Но строка ведёт себя похоже на массив символов, потому что позволяет обращаться к символам по индексу.

---

## Пример

```csharp
string line = "Sdsdsd";

Console.WriteLine(line[0]);
Console.WriteLine(line[1]);
Console.WriteLine(line[2]);
```

Результат:

```text
S
d
s
```

---

## Индексация

Как и в массивах, индексация начинается с нуля:

```text
Строка:  S  d  s  d  s  d
Индекс:  0  1  2  3  4  5
```

---

## Длина строки

```csharp
string line = "Sdsdsd";

Console.WriteLine(line.Length);
```

Результат:

```text
6
```

---

## Перебор строки через `foreach`

```csharp
string line = "Hello";

foreach (char symbol in line)
{
    Console.WriteLine(symbol);
}
```

Результат:

```text
H
e
l
l
o
```

---

## Почему строка не массив?

Массив:

```csharp
char[] letters = { 'H', 'e', 'l', 'l', 'o' };

letters[0] = 'X'; // можно
```

Строка:

```csharp
string text = "Hello";

// text[0] = 'X'; // Ошибка
```

Строки в C# **неизменяемы (immutable)**.

После создания нельзя изменить отдельный символ.

Чтобы изменить строку, создаётся новая:

```csharp
string text = "Hello";
text = "Xello";
```

---

## Преобразование строки в массив символов

```csharp
string text = "Hello";

char[] chars = text.ToCharArray();
```

Теперь:

```csharp
chars[0] = 'X';
```

можно изменять элементы массива.

---

## Итог

- `string` — это класс, а не массив.
- Можно обращаться к символам через `[]`.
- Есть свойство `Length`.
- Можно перебирать через `foreach`.
- В отличие от массива, строка неизменяема.
