## Коллекция `List<T>`

`List<T>` — это динамический массив.

В отличие от обычного массива (`int[]`), список может автоматически увеличивать и уменьшать свой размер.

---

## Создание списка

```csharp
List<int> numbers = new List<int>();
```

или

```csharp
List<int> numbers = new List<int>(5);
```

Число `5` — это начальная вместимость (*capacity*), а не количество элементов.

---

## Добавление элементов

```csharp
numbers.Add(12);
```

Добавляет элемент в конец списка.

---

## Добавление нескольких элементов

```csharp
numbers.AddRange(new int[] { 3, 4, 5, 6 });
```

Результат:

```text
[12, 3, 4, 5, 6]
```

---

## Удаление по индексу

```csharp
numbers.RemoveAt(3);
```

Удаляет элемент с индексом `3`.

Все элементы справа сдвигаются влево.

---

## Удаление по значению

```csharp
numbers.Remove(4);
```

Удаляет первое найденное значение `4`.

---

## Очистка списка

```csharp
numbers.Clear();
```

Удаляет все элементы.

---

## Поиск индекса

```csharp
numbers.IndexOf(12);
```

Возвращает индекс первого найденного элемента.

Если элемент не найден:

```text
-1
```

---

## Вставка по индексу

```csharp
numbers.Insert(1, 123);
```

Было:

```text
[12, 3, 5, 6]
```

Стало:

```text
[12, 123, 3, 5, 6]
```

Все элементы после индекса `1` сдвигаются вправо.

---

## Количество элементов

```csharp
numbers.Count
```

Возвращает текущее количество элементов в списке.

Аналог:

```csharp
array.Length
```

для массива.

---

## Перебор списка

Через `for`:

```csharp
for (int i = 0; i < numbers.Count; i++)
{
    Console.WriteLine(numbers[i]);
}
```

---

Через `foreach`:

```csharp
foreach (int number in numbers)
{
    Console.WriteLine(number);
}
```

---

## Отличие массива от списка

Массив:

```csharp
int[] nums = new int[5];
```

Размер фиксирован.

---

Список:

```csharp
List<int> nums = new List<int>();
```

Размер изменяется автоматически.

---

## Когда использовать?

✅ Массив (`int[]`)
- размер известен заранее;
- максимальная производительность.

✅ Список (`List<T>`)
- элементы часто добавляются или удаляются;
- размер заранее неизвестен.

---

## Часто используемые методы

```csharp
Add()        // добавить элемент
AddRange()   // добавить несколько элементов
Remove()     // удалить по значению
RemoveAt()   // удалить по индексу
Insert()     // вставить по индексу
IndexOf()    // найти индекс
Clear()      // очистить список
Contains()   // проверить наличие элемента
Count        // количество элементов
```
