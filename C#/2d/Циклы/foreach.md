## Цикл `foreach`

Цикл `foreach` используется для **перебора всех элементов коллекции** (массива, списка, строки и т.д.).

---

## Пример через `for`

```csharp
int[] nums = { 0, 1, 2, 3, 4, 5 };

for (int i = 0; i < nums.Length; i++)
{
    Console.WriteLine(nums[i]);
}
```

Здесь мы используем индекс:

```csharp
nums[i]
```

чтобы получить элемент массива.

---

## Тот же пример через `foreach`

```csharp
foreach (int i in nums)
{
    Console.WriteLine(i);
}
```

Результат:

```text
0
1
2
3
4
5
```

---

## Как читается?

```csharp
foreach (int i in nums)
```

означает:

> Для каждого элемента массива `nums` сохранить его в переменную `i` и выполнить код внутри цикла.

---

## Что происходит?

Массив:

```text
[0][1][2][3][4][5]
```

Итерации:

```text
i = 0
i = 1
i = 2
i = 3
i = 4
i = 5
```

---

## Преимущества `foreach`

- код короче;
- не нужен счётчик `i`;
- нельзя случайно выйти за границы массива;
- удобно, когда нужен только элемент.

---

## Ограничения `foreach`

Внутри `foreach` нет индекса элемента:

```csharp
foreach (int item in nums)
{
    Console.WriteLine(item);
}
```

Поэтому нельзя узнать номер элемента напрямую.

Если нужен индекс, лучше использовать `for`.

---

## Когда использовать?

✅ `foreach`

Когда нужно просто перебрать все элементы:

```csharp
foreach (string name in names)
{
    Console.WriteLine(name);
}
```

✅ `for`

Когда нужен индекс:

```csharp
for (int i = 0; i < names.Length; i++)
{
    Console.WriteLine($"{i}: {names[i]}");
}
```

---

## Кратко

```text
for     → есть индекс и полный контроль
foreach → удобный перебор всех элементов
```
