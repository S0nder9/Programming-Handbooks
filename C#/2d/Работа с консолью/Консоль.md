## Полезные методы и свойства `Console`

### Специальные символы

```csharp
Console.WriteLine("Hello\nWorld");
```

`\n` — перенос строки.

Результат:

```text
Hello
World
```

---

```csharp
Console.WriteLine("Helloo\b");
```

`\b` — удаляет предыдущий символ (Backspace).

Результат:

```text
Hello
```

---

## Очистка консоли

```csharp
Console.Clear();
```

Полностью очищает окно консоли.

---

## Установка позиции курсора

```csharp
Console.SetCursorPosition(5, 10);
```

Перемещает курсор:
- на 5 символов вправо;
- на 10 строк вниз.

После этого следующий вывод начнётся из этой позиции.

---

## Изменение цвета текста

```csharp
Console.ForegroundColor = ConsoleColor.Green;
```

Меняет цвет текста на зелёный.

Пример:

```csharp
Console.ForegroundColor = ConsoleColor.Green;
Console.WriteLine("Hello");
```

---

## Изменение цвета фона

```csharp
Console.BackgroundColor = ConsoleColor.Red;
```

Меняет цвет фона.

Чтобы изменения стали видны для всей консоли:

```csharp
Console.BackgroundColor = ConsoleColor.Red;
Console.Clear();
```

---

## Размер окна консоли

```csharp
Console.WindowHeight = 10;
Console.WindowWidth = 20;
```

Устанавливает:
- высоту окна в 10 строк;
- ширину окна в 20 символов.

---

## Сброс цветов

После изменения цветов полезно вернуть стандартные настройки:

```csharp
Console.ResetColor();
```

Пример:

```csharp
Console.ForegroundColor = ConsoleColor.Green;
Console.WriteLine("Green text");

Console.ResetColor();
Console.WriteLine("Default text");
```

---

## Важно

Метод называется:

```csharp
Console.SetCursorPosition(...)
```

а не:

```csharp
Console.setCursorPosition(...)
```

В C# имена методов чувствительны к регистру.
