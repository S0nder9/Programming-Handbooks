## Конвертация

```csharp
int age;
string input = "15";
age = Convert.ToInt32(input);
Console.WriteLine(age);

float res;
int x = 5;
int y = 2;

res = Convert.ToSingle(x) / y;

Console.WriteLine(res);
```
