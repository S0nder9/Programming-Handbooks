## Конкатенация

```csharp
string greeting = "Hello, World!";
string name = "Test";
string message = greeting + " " + name;

Console.WriteLine(message + ", !"); // Hello, World! Test, !

int age = 5;
Console.WriteLine("Hi! " + age + 1); // Hi! 51: знак + выполняет конкатенацию строк, а не сложение чисел, поэтому 51, а не 6
Console.WriteLine("Hi! " + (age + 1)); // Hi! 6: скобки изменяют порядок выполнения операций, сначала выполняется сложение чисел, а затем конкатенация строк
Console.ReadKey();
```

## Соединение строк:

```csharp
int age = 27;
string name = "John";

Console.WriteLine("Name: " + name + "Df" + age + "sds");
Console.WriteLine($"Name: {name} df {age} sds"); // Интерполяция строк
```
