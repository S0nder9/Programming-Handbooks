## Переменные -

это контейнеры для хранения информации.

`Объясвление переменной`

int money;

`Инициализация переменной`

money = 100;

---

## Описание переменной

## Тип данным, имя переменной, =, значение переменной

## Типы данных:

1. Целочисленные:

```csharp
            byte b; // от 0 до 255
            sbyte sb; // от -128 до 127
            short s; // от -32768 до 32767
            ushort su; // от 0 до 65535
            int i; // от -2147483648 до 2147483647
            uint ui; // от 0 до 4294967295
            long l = long.MaxValue; // Узнаем максимальное значение типа long
            long m = long.MinValue; // Узнаем минимальное значение типа
```

2. Числа с плавающей точкой:

```csharp
            float f; // точность значения 7 знаков после запятой
            double d; // точность значения 15-16 знаков после запятой

            float f1 = 5.7f; // для указания нужно использовать суффикс f
            double d1 = 5.7;
```

3. Символьный тип:

```csharp
           char c = 'T'; // для указания символа нужно использовать одинарные кавычки
```

4. Строковый тип:

```csharp
           string s = "Hello world!"; // для указания строки нужно использовать двойные кавычки
```

5. Логический тип:

```csharp
            bool bl; // логический тип данных, может принимать только два значения: true и false

```
