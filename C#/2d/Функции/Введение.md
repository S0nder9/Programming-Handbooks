## Функции (Методы)

**Функция (метод)** — это именованный блок кода, который можно вызывать много раз.

Функции помогают:
- не дублировать код;
- делать программу более понятной;
- разбивать большую задачу на маленькие части.

> В C# функции внутри классов обычно называют **методами**.

---

## Пример встроенной функции

```csharp
string userInput = "SDsdsd";

userInput.ToLower();
```

Метод `ToLower()` переводит строку в нижний регистр:

```text
SDsdsd → sdsdsd
```

---

## Собственная функция без параметров

```csharp
static void WriteError()
{
    Console.WriteLine("123");
}
```

Вызов:

```csharp
WriteError();
```

Результат:

```text
123
```

---

## Функция с параметром

```csharp
static void WriteError2(string text)
{
    Console.WriteLine(text);
}
```

Вызов:

```csharp
WriteError2("123");
```

Результат:

```text
123
```

---

## Параметр и аргумент

Часто их путают.

### Параметр

Переменная в объявлении функции:

```csharp
static void WriteError2(string text)
{
}
```

Здесь:

```csharp
string text
```

— **параметр**.

---

### Аргумент

Значение, которое передаётся при вызове функции:

```csharp
WriteError2("123");
```

Здесь:

```csharp
"123"
```

— **аргумент**.

---

## Разбор метода

```csharp
static void WriteError2(string text)
```

Состоит из:

- `static` — метод принадлежит классу;
- `void` — ничего не возвращает;
- `WriteError2` — имя метода;
- `string text` — параметр.

---

## Метод с возвращаемым значением

```csharp
static int Sum(int a, int b)
{
    return a + b;
}
```

Вызов:

```csharp
int result = Sum(5, 3);

Console.WriteLine(result);
```

Результат:

```text
8
```

---

## Кратко

```text
Параметр → переменная в объявлении функции
Аргумент → значение при вызове функции

void → ничего не возвращает
return → возвращает результат
```
