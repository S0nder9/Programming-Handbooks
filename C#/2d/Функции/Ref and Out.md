## Ключевые слова `ref` и `out`

`ref` и `out` позволяют передавать переменные **по ссылке**, то есть метод может изменить значение переменной вне себя.

---

## `ref`

Переменная должна быть **инициализирована до вызова метода**.

```csharp
int sum = 0;
int x = 1;
int y = 5;

Add(ref sum, x, y);
```

Метод:

```csharp
static void Add(ref int sum, int x, int y)
{
    sum = x + y;
}
```

Результат:

```text
6
```

---

## `out`

Переменную можно не инициализировать до вызова.

```csharp
int sum;
int x = 1;
int y = 5;

Add(out sum, x, y);
```

Метод обязан присвоить значение параметру `out`:

```csharp
static void Add(out int sum, int x, int y)
{
    sum = x + y;
}
```

Иначе будет ошибка компиляции.

---

## Разница между `ref` и `out`

```text
ref → переменная должна иметь значение до вызова

out → переменная может быть неинициализированной
```

---

## Массивы и ссылочные типы

```csharp
int[] array = new int[5];
```

Массив является ссылочным типом.

Поэтому такой метод работает:

```csharp
static void EditArray(int[] array, int index, int value)
{
    array[index] = value;
}
```

Вызов:

```csharp
EditArray(array, 2, 5);
```

Результат:

```csharp
Console.WriteLine(array[2]); // 5
```

Потому что изменяется сам объект массива.

---

## Почему второй пример не работает?

```csharp
static int[] EditArray(int[] array, int index, int value)
{
    array = new int[6];
    array[index] = value;

    return array;
}
```

Здесь:

```csharp
array = new int[6];
```

создаёт **новый массив**.

Изменяется только локальная переменная `array` внутри метода.

Исходный массив в `Main` остаётся прежним.

---

## Как получить новый массив?

Нужно сохранить результат:

```csharp
array = EditArray(array, 2, 5);
```

Теперь переменная `array` будет ссылаться на новый массив размером 6.

---

## Альтернатива через `ref`

Можно изменить саму ссылку:

```csharp
static void EditArray(ref int[] array)
{
    array = new int[6];
}
```

Вызов:

```csharp
EditArray(ref array);
```

После выхода из метода переменная `array` будет ссылаться на новый массив.

---

## Когда использовать?

- `ref` — если нужно прочитать и изменить существующее значение.
- `out` — если метод должен вернуть дополнительный результат.
- Для массивов обычно `ref` не нужен, если меняются только элементы массива.
