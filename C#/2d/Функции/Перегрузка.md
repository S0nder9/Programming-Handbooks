## Перегрузка методов (Method Overloading)

**Перегрузка** — это создание нескольких методов с одинаковым именем, но разными параметрами.

Компилятор сам определяет, какой метод вызвать, по количеству и типам аргументов.

---

## Пример

В программе есть два метода:

```csharp
static int[] Resize(int[] array, int size)
```

и

```csharp
static int[,] Resize(int[,] array, int x, int y)
```

Название одинаковое:

```csharp
Resize(...)
```

но параметры разные.

---

## Вызов метода для одномерного массива

```csharp
int[] array1 = new int[5];

array1 = Resize(array1, 6);
```

Компилятор видит:

```csharp
int[]
```

и вызывает:

```csharp
static int[] Resize(int[] array, int size)
```

---

## Вызов метода для двумерного массива

```csharp
int[,] array2 = new int[5, 5];

array2 = Resize(array2, 10, 10);
```

Компилятор видит:

```csharp
int[,]
```

и вызывает:

```csharp
static int[,] Resize(int[,] array, int x, int y)
```

---

## Зачем нужна перегрузка?

Чтобы использовать одно логичное имя для похожих действий.

Например:

```csharp
Resize(...)
```

означает:

> "Изменить размер массива"

независимо от того, одномерный он или двумерный.

---

## Что можно менять при перегрузке?

✅ Количество параметров:

```csharp
Print(string text)
Print(string text, int count)
```

---

✅ Типы параметров:

```csharp
Sum(int a, int b)
Sum(double a, double b)
```

---

✅ Порядок типов:

```csharp
Test(int x, string y)
Test(string y, int x)
```

---

## Что нельзя?

Нельзя перегружать метод только по возвращаемому типу:

❌ Ошибка:

```csharp
int Sum(int a, int b)
double Sum(int a, int b)
```

Параметры одинаковые, поэтому компилятор не сможет понять, какой метод вызывать.

---

## Итог

```text
Перегрузка = несколько методов с одним именем,
но разными параметрами.
```

Это делает код удобнее и понятнее.
