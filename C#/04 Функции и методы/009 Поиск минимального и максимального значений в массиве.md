## Поиск минимального и максимального значений в массиве

В массиве можно найти:
- минимальный элемент (`min`)
- максимальный элемент (`max`)

Для этого обычно используют **перебор элементов через цикл**.

---

## Алгоритм

1. Берём первый элемент массива как стартовое значение.
2. Сравниваем его с остальными.
3. Обновляем `min` или `max`, если нашли больше/меньше.

---

## Пример массива

```csharp
int[] numbers = { 5, 2, 9, 1, 7 };
```

---

## Поиск максимума

```csharp
int max = numbers[0];

for (int i = 1; i < numbers.Length; i++)
{
    if (numbers[i] > max)
    {
        max = numbers[i];
    }
}

Console.WriteLine("Max: " + max);
```

Результат:
```text
Max: 9
```

---

## Поиск минимума

```csharp
int min = numbers[0];

for (int i = 1; i < numbers.Length; i++)
{
    if (numbers[i] < min)
    {
        min = numbers[i];
    }
}

Console.WriteLine("Min: " + min);
```

Результат:
```text
Min: 1
```

---

## Почему начинаем с `numbers[0]`?

Потому что:
- массив точно не пустой (в задаче обычно так);
- это даёт корректную стартовую точку сравнения.

---

## Одновременный поиск min и max

Можно сразу найти оба:

```csharp
int[] numbers = { 5, 2, 9, 1, 7 };

int min = numbers[0];
int max = numbers[0];

for (int i = 1; i < numbers.Length; i++)
{
    if (numbers[i] < min)
        min = numbers[i];

    if (numbers[i] > max)
        max = numbers[i];
}

Console.WriteLine("Min: " + min);
Console.WriteLine("Max: " + max);
```

Результат:
```text
Min: 1
Max: 9
```
