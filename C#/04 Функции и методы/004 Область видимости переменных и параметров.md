## Область видимости переменных и параметров в функциях и методах

**Область видимости (scope)** — это часть программы, где переменная или параметр **существует** и доступен для использования.

В функциях и методах переменные бывают:
- **локальные переменные**;
- **параметры метода**.

---

## Локальные переменные

Переменная, объявленная внутри метода, доступна **только внутри этого метода**.

```csharp
static void MyMethod()
{
    int x = 10;
    Console.WriteLine(x); // работает
}

// Console.WriteLine(x); // Ошибка: x не существует вне метода
```

После завершения метода переменная уничтожается.

---

## Параметры метода

Параметры — это переменные, передаваемые в метод.

```csharp
static void SayHello(string name)
{
    Console.WriteLine($"Hello, {name}");
}
```

Здесь `name` — параметр.

Он существует **только внутри метода**:

```csharp
SayHello("Nikita");

// Console.WriteLine(name); // Ошибка
```

---

## Локальная переменная и параметр

Можно использовать параметр как обычную переменную:

```csharp
static void AddOne(int number)
{
    number++;
    Console.WriteLine(number);
}
```

Вызов:

```csharp
AddOne(5);
```

Результат:
```text
6
```

Но исходное значение вне метода не изменится (если не использовать `ref`).

---

## Переменные внутри блока в методе

Если внутри метода есть `if`, `for`, `while`, то переменные внутри них живут только в своём блоке:

```csharp
static void Test()
{
    if (true)
    {
        int y = 20;
        Console.WriteLine(y); // работает
    }

    // Console.WriteLine(y); // Ошибка
}
```

---

## Вложенные блоки видят внешние переменные

```csharp
static void Test()
{
    int x = 5;

    if (true)
    {
        Console.WriteLine(x); // 5
    }
}
```

Внутренний блок видит внешнюю переменную.

---

## Затенение параметра запрещено

Так делать нельзя:

```csharp
static void Test(int x)
{
    int x = 10; // Ошибка
}
```

Почему?  
Потому что параметр `x` уже существует в области видимости метода.

---

## Разные методы — разные области видимости

Переменные из одного метода недоступны в другом:

```csharp
static void Method1()
{
    int a = 10;
}

static void Method2()
{
    // Console.WriteLine(a); // Ошибка
}
```

Каждый метод имеет свою собственную область видимости.

---

## Правило

Переменная или параметр живёт:
- с момента объявления;
- до конца блока `{ }`, в котором объявлена.
