## Поля, переменные экземпляра и их отличие от локальных

```csharp
namespace OldStyle
{
    internal class Program
    {

        int myResult; // объявляем переменную на уровне класса

        static void Main(string[] args)
        {
            myResult = Two(5); // вызываем метод и сохраняем результат в переменную

            Console.WriteLine(myResult); 
        }

        static int Two(int number1)
        {
            myResult = number1 * 2;
            return myResult; 
        }
    }
}```
