# Условный оператор `if` и логические операторы

## Условный оператор `if`

`if` используется для выполнения кода **при выполнении условия**.

Синтаксис:

```csharp
if (условие)
{
    // код выполнится, если условие true
}
```

Пример:

```csharp
int age = 18;

if (age >= 18)
{
    Console.WriteLine("Доступ разрешён");
}
```

Результат:
```text
Доступ разрешён
```

---

## `if ... else`

Если условие ложное (`false`), выполнится блок `else`.

```csharp
int age = 16;

if (age >= 18)
{
    Console.WriteLine("Доступ разрешён");
}
else
{
    Console.WriteLine("Доступ запрещён");
}
```

Результат:
```text
Доступ запрещён
```

---

## `if ... else if ... else`

Используется для нескольких условий.

```csharp
int score = 85;

if (score >= 90)
{
    Console.WriteLine("Отлично");
}
else if (score >= 70)
{
    Console.WriteLine("Хорошо");
}
else
{
    Console.WriteLine("Нужно подтянуть");
}
```

Результат:
```text
Хорошо
```

# Операторы сравнения

| Оператор | Значение |
|----------|----------|
| `==` | равно |
| `!=` | не равно |
| `>` | больше |
| `<` | меньше |
| `>=` | больше или равно |
| `<=` | меньше или равно |

Пример:

```csharp
int x = 10;

Console.WriteLine(x == 10); // True
Console.WriteLine(x != 5);  // True
Console.WriteLine(x > 3);   // True


int num1 = 5;
int num2 = 6;

bool isHigher = num1 > num2; // False

```

# Логические операторы

## `&&` (И)

Истина только если **оба условия истинны**.

```csharp
int age = 20;
bool hasTicket = true;

if (age >= 18 && hasTicket)
{
    Console.WriteLine("Можно войти");
}
```

---

## `||` (ИЛИ)

Истина, если **хотя бы одно условие истинно**.

```csharp
int age = 16;
bool withParent = true;

if (age >= 18 || withParent)
{
    Console.WriteLine("Можно войти");
}
```

---

## `!` (НЕ)

Меняет `true` на `false` и наоборот.

```csharp
bool isRain = false;

if (!isRain)
{
    Console.WriteLine("Можно гулять");
}
```

Результат:
```text
Можно гулять
```

---

# Вложенные `if`

```csharp
int age = 20;
bool hasPassport = true;

if (age >= 18)
{
    if (hasPassport)
    {
        Console.WriteLine("Проход разрешён");
    }
}
```

---

# Тернарный оператор `?`

Короткая запись `if...else`.

```csharp
int age = 20;

string result = age >= 18 ? "Взрослый" : "Ребёнок";
Console.WriteLine(result);
```

Результат:
```text
Взрослый
```
