## Трёхмерные массивы

**Трёхмерный массив** — это массив, который имеет **3 измерения**.

Его можно представить как **несколько двумерных таблиц (слоёв)**, сложенных друг на друга, как куб.

Например:

```text
Слой 0:
[1] [2]
[3] [4]

Слой 1:
[5] [6]
[7] [8]
```

Каждый элемент определяется **тремя индексами**:

```csharp
array[слой, строка, столбец]
```

Например:

```csharp
array[0, 1, 0]
```

означает:
- слой `0` (первый слой),
- строка `1` (вторая строка),
- столбец `0` (первый столбец).

---

## Создание трёхмерного массива

```csharp
int[,,] array3D = new int[2, 2, 2];
```

Создаётся массив:
- 2 слоя,
- в каждом слое 2 строки,
- в каждой строке 2 столбца.

По умолчанию:

```text
Слой 0:
[0] [0]
[0] [0]

Слой 1:
[0] [0]
[0] [0]
```

---

## Инициализация значениями

```csharp
int[,,] array3D =
{
    {
        { 1, 2 },
        { 3, 4 }
    },
    {
        { 5, 6 },
        { 7, 8 }
    }
};
```

Получаем:

```text
Слой 0:
[1] [2]
[3] [4]

Слой 1:
[5] [6]
[7] [8]
```

---

## Доступ к элементу

```csharp
Console.WriteLine(array3D[0, 1, 1]);
```

Результат:
```text
4
```

Пояснение:
- слой `0`
- строка `1`
- столбец `1`

→ число `4`

---

## Изменение элемента

```csharp
array3D[1, 0, 1] = 10;

Console.WriteLine(array3D[1, 0, 1]);
```

Результат:
```text
10
```

---

## Размер массива

Получить количество элементов по каждой размерности:

```csharp
array3D.GetLength(0); // количество слоёв
array3D.GetLength(1); // количество строк
array3D.GetLength(2); // количество столбцов
```

Пример:

```csharp
Console.WriteLine(array3D.GetLength(0)); // 2
Console.WriteLine(array3D.GetLength(1)); // 2
Console.WriteLine(array3D.GetLength(2)); // 2
```

---

## Перебор трёхмерного массива

Используются **три вложенных цикла**:

```csharp
for (int i = 0; i < array3D.GetLength(0); i++)
{
    for (int j = 0; j < array3D.GetLength(1); j++)
    {
        for (int k = 0; k < array3D.GetLength(2); k++)
        {
            Console.WriteLine(array3D[i, j, k]);
        }
    }
}
```

Вывод:
```text
1
2
3
4
5
6
7
8
```
