## Неявное преобразование типов

```csharp
int myInt = 13;
double myDouble = myInt; // Неявное преобразование (не требует явного указания)

long myLong = myInt;

myInt = myLong; // Ошибка компиляции: невозможно неявно преобразовать long в int

float = myFloat = 123.123f; // Явное преобразование (требует суффикса 'f' для указания типа float)
myDouble = myFloat; // Неявное преобразование (float может быть неявно преобразован в double)
```

## Явное преобразование типов

```csharp

long longNum = 280214242141241;
int myInt2 = (int)longNum; // Явное преобразование (требует указания типа в скобках), но может привести к потере данных, так как long может содержать больше данных, чем int.

float floatNum = 123.456123213321f;
double doubleNum = 123.456123213321;
floatNum = (float)doubleNum; // Потеря данных: 123,45612

Console.WriteLine(floatNum);

Console.ReadLine();

```
