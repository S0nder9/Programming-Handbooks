## Инкремент (`++`) и декремент (`--`)

Инкремент (`++`) увеличивает значение переменной на **1**.

```csharp
int num = 0;

Console.WriteLine("Num is: " + num); // num = 0

num++; // Постфиксный инкремент

Console.WriteLine("Num is: " + num); // num = 1

++num; // Префиксный инкремент

Console.WriteLine("Num is: " + num); // num = 2
```

Когда `++` стоит **отдельной строкой**, разницы нет:

```csharp
num++;
++num;
```

Оба варианта увеличат `num` на 1.

---

## Разница между `num++` и `++num`

Разница появляется, когда оператор используется **в выражении**.

```csharp
num = 0;

Console.WriteLine("Num is: {0}", num);   // 0
Console.WriteLine("Num is: {0}", ++num); // 1
Console.WriteLine("Num is: {0}", num++); // 1
Console.WriteLine("Num is: {0}", num);   // 2
```

---

### `++num` (префиксный)

Сначала увеличивает, потом использует значение.

```csharp
++num;
```

Это то же самое, что:

```csharp
num = num + 1;
Console.WriteLine(num);
```

Пример:

```csharp
num = 0;
Console.WriteLine(++num);
```

Шаги:
1. `num = 0`
2. увеличить → `num = 1`
3. вывести → `1`

Результат:
```text
1
```

---

### `num++` (постфиксный)

Сначала использует текущее значение, потом увеличивает.

```csharp
num++;
```

Это примерно как:

```csharp
Console.WriteLine(num);
num = num + 1;
```

Пример:

```csharp
num = 0;
Console.WriteLine(num++);
```

Шаги:
1. `num = 0`
2. вывести → `0`
3. увеличить → `num = 1`

Результат:
```text
0
```

## Пример в присваивании

```csharp
int a = 5;
int b = ++a;
```

После выполнения:
```text
a = 6
b = 6
```

Потому что сначала увеличили `a`.

---

```csharp
int a = 5;
int b = a++;
```

После выполнения:
```text
a = 6
b = 5
```

Потому что сначала взяли старое значение `a`, потом увеличили.

---

## Декремент (`--`)

Работает так же, только уменьшает на 1:

```csharp
int x = 5;

x--; // 4
--x; // 3
```

---

## Увеличение/Уменьшене значение не на 1, а но определенное значение:

```csharp
int num = 0;

num += 30;
num -= 10;

num *= 30;
num /= 10;
```
