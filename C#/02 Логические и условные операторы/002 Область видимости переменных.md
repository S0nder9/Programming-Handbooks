## Область видимости переменных

**Область видимости (scope)** — это часть программы, где переменная **существует** и доступна для использования.

---

### Локальная переменная

Переменная, объявленная внутри блока `{ }`, доступна **только внутри этого блока**.

```csharp
if (true)
{
    int x = 10;
    Console.WriteLine(x); // работает
}

Console.WriteLine(x); // Ошибка: x не существует вне блока
```

---

### Переменные внутри `if`

```csharp
if (true)
{
    string message = "Hello";
    Console.WriteLine(message); // работает
}

// Console.WriteLine(message); // Ошибка
```

После выхода из `if` переменная уничтожается.

---

### Вложенные блоки

Внутренний блок видит переменные внешнего блока:

```csharp
int a = 5;

if (true)
{
    Console.WriteLine(a); // 5
}
```

Но внешний блок **не видит** внутренние:

```csharp
if (true)
{
    int b = 10;
}

// Console.WriteLine(b); // Ошибка
```

---

### Переменные в циклах

Переменная цикла существует только внутри цикла:

```csharp
for (int i = 0; i < 3; i++)
{
    Console.WriteLine(i);
}

// Console.WriteLine(i); // Ошибка
```

---

### Область видимости метода

Переменные внутри метода доступны только в этом методе.

```csharp
static void MyMethod()
{
    int x = 100;
    Console.WriteLine(x);
}

// Console.WriteLine(x); // Ошибка
```

---

### Глобальные (поля класса)

Переменные, объявленные внутри класса, но вне методов, доступны всем методам класса.

```csharp
class Program
{
    static int age = 18;

    static void Main()
    {
        Console.WriteLine(age);
    }
}
```

Здесь `age` — **поле класса**, а не локальная переменная.

---

## Shadowing (затенение переменной)

Можно создать переменную с тем же именем во внутреннем блоке — внешняя временно "скрывается".

```csharp
int x = 5;

{
    int x = 10; // Ошибка в C#: нельзя объявить в той же вложенности
}
```

В C# так делать нельзя в пересекающихся областях видимости.

---

## Правило

Переменная живёт:
- с момента объявления;
- до закрывающей `}` своего блока.

```csharp
{
    int number = 42;
} // здесь number уничтожается
```
