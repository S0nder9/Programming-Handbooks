## Типы значений (Value Types)

Типы значений хранят **само значение** прямо в памяти (обычно в стеке, stack).

При передаче в функцию создаётся **копия значения**, поэтому изменение внутри метода **не влияет** на исходную переменную.

Примеры:
- int
- double
- float
- bool
- char
- decimal

Также к типам значений относятся:
- struct (структуры)
- enum (перечисления)

Особенности:
- не могут быть `null` по умолчанию (кроме nullable: `int?`, `bool?`)
- работают быстрее, так как не требуют выделения памяти в куче

Пример:
```csharp
int a = 5;
int b = a;
b = 10;

// a = 5, потому что была создана копия
```

## Ссылочные типы (Reference Types)

Ссылочные типы `хранят ссылку` (адрес) `на объект в памяти` (в куче, heap), а не сам объект.

При передаче в функцию копируется ссылка, поэтому несколько переменных могут указывать на один и тот же объект.

Если изменить объект через одну переменную — изменения увидят все ссылки на этот объект.

Примеры:

`string`
`class`
`array`
`interface`
`delegate`
`object`

`Особенности`:

- могут быть null
- хранятся в куче (heap)
- требуют сборщика мусора (Garbage Collector)

Пример:
```csharp
int[] arr1 = { 1, 2, 3 };
int[] arr2 = arr1;

arr2[0] = 100;

// arr1[0] тоже станет 100,
// потому что обе переменные указывают на один объект

```

## Важное исключение: string

`string` — это ссылочный тип, но ведёт себя похоже на тип значения, потому что неизменяемый (immutable).

Пример:

string a = "Hello";
string b = a;

b = "World";

// a останется "Hello"

Создаётся новый объект, старый не изменяется.
