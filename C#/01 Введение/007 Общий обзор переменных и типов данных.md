Переменные в C#

`Переменные` — контейнер для хранения значений.  
По умолчанию переменная является **изменяемой**.

У переменной должен быть **тип** и **имя**.

## Синтаксис объявления

`1. Основные категории типов в C#`

В C# все типы делятся на две большие группы:
Value Types (Типы-значения)
Хранят само значение прямо в стеке (или в куче внутри объекта). Копируется по значению.

- **Целочисленные типы**
  - `sbyte` (8-bit, -128..127)
  - `byte` (8-bit, 0..255)
  - `short` (16-bit)
  - `ushort` (16-bit)
  - `int` (32-bit) ← самый популярный
  - `uint` (32-bit)
  - `long` (64-bit)
  - `ulong` (64-bit)
  - `nint` / `nuint` (native int, зависит от платформы)

- **Числа с плавающей точкой**
  - `float` (32-bit, одинарная точность)
  - `double` (64-bit, двойная точность) ← по умолчанию для литералов
  - `decimal` (128-bit, финансовые расчёты)

- **Другие встроенные**
  - `bool` (true/false)
  - `char` (16-bit Unicode)
  - `string` — **reference type**, но ведёт себя как value (иммутабельный)

- **Структура (`struct`)**
- **Перечисление (`enum`)**
- **Nullable value types** (`T?` или `Nullable<T>`)

Reference Types (Ссылочные типы)
Хранят ссылку на объект в куче. Копируется по ссылке.

- `class` (ссылочный тип)
- `interface`
- `delegate`
- `object` (базовый тип для всех)
- `string`
- `dynamic`
- Массивы (`int[]`, `string[]` и т.д.)
- `record` (C# 9+) — reference type с value-подобным поведением
- `record struct` (C# 10+) — value type


2. Специальные модификаторы типов

`const` — константа, значение должно быть известно на компиляции
`readonly` — можно установить только в конструкторе или при объявлении
`ref` — передача по ссылке
`in` — readonly ссылка (C# 7.2)
`out` — выходной параметр
`nullable reference types (string?)` — включено через <Nullable>enable</Nullable>

3. Виды переменных по области видимости и назначению

1. **Локальные переменные** — внутри метода
2. **Поля класса** (instance fields)
3. **Статические поля** (`static`)
4. **Константы** (`const`)
5. **Свойства** (`Property`)
6. **Параметры метода**
7. **ref/out/in параметры**
8. **Локальные функции** и **лямбды** (замыкания)
9. **Tuple** (`(int, string)`)
10. **Record**
