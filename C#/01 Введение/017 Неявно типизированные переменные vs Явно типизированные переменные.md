## Неявно типизированные переменные (`var`)

Компилятор сам определяет тип переменной на основе присвоенного значения.

```csharp
var myFavouriteGenre = "Sci-fi"; // myFavouriteGenre имеет тип string
var myFavouriteNumber = 42;      // myFavouriteNumber имеет тип int
var price = 19.99;               // price имеет тип double
var isActive = true;             // isActive имеет тип bool
```

Особенности:
- тип определяется только **при инициализации**;
- после определения тип изменить нельзя;
- нельзя написать `var x;` без значения.

Пример ошибки:
```csharp
var x; // Ошибка: невозможно определить тип
```

---

## Явно типизированные переменные

Разработчик сам указывает тип переменной.

```csharp
string myFavouriteGenre = "Sci-fi";
int myFavouriteNumber = 42;
double price = 19.99;
bool isActive = true;
```

Преимущества:
- код более читаемый;
- сразу видно, какой тип у переменной;
- удобно для больших проектов.

---

## Когда использовать `var`

Использовать `var` удобно, когда тип очевиден:

```csharp
var name = "Nikita";   // понятно, что это string
var age = 18;          // понятно, что это int
```

Не очень удобно:

```csharp
var data = GetUserData(); // неясно, какой тип вернёт метод
```
