# Ввод пользователем текста

```csharp
string userInput = Console.ReadLine(); // Используйте метод Console.ReadLine() для получения ввода от пользователя и сохраните его в переменной userInput.
Console.WriteLine("You entered: " + userInput);
Console.ReadKey(); // Используйте метод Console.ReadKey() для ожидания нажатия клавиши пользователем, чтобы программа не закрывалась сразу после вывода результата.
```
