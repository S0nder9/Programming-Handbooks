## - escape-последовательности в строках

```csharp
string s1 = "thidfsdnj \"ionjio\" a / \n and :  and \\";
// Воспринимает кавычки как часть строки, а не как ограничители
// \n - это символ новой строки, который будет отображаться как перенос строки в выводе

Console.WriteLine(s1);

Console.ReadKey();
```
