## Методы в классах

**Метод** — это функция, которая принадлежит классу и описывает **поведение объекта** (что объект умеет делать).
`Функция не является частью класса, а является частью объекта. Метод же является частью класса.
`
Например:
- автомобиль может `Drive()`,
- человек может `Walk()`,
- собака может `Bark()`.

---

## Пример метода

```csharp
public void Drive()
{
    Console.WriteLine($"{Name} is driving.");
}
```

Разберём:

- `public` — метод доступен извне класса;
- `void` — метод **ничего не возвращает**;
- `Drive` — имя метода;
- `()` — параметры метода (здесь параметров нет).

---

## Вызов метода

```csharp
Customer customer =
    new Customer("John Doe", "123 Main St", "555-1234");

customer.Drive();
```

Результат:

```text
John Doe is driving.
```

---

## Как это работает?

Когда вызывается:

```csharp
customer.Drive();
```

объект `customer` вызывает свой метод `Drive()`.

Внутри метода используется свойство:

```csharp
Name
```

поэтому выводится имя именно этого объекта.

---

## Метод с параметрами

Метод может принимать параметры:

```csharp
public void SayHello(string message)
{
    Console.WriteLine($"{Name}: {message}");
}
```

Вызов:

```csharp
customer.SayHello("Hello!");
```

Результат:

```text
John Doe: Hello!
```

---

## Метод с возвращаемым значением

Если нужно вернуть результат:

```csharp
public string GetInfo()
{
    return $"{Name}, {Address}";
}
```

Использование:

```csharp
string info = customer.GetInfo();
Console.WriteLine(info);
```

---

## Идея

```text
Свойства = что объект хранит
Методы = что объект умеет делать
```

Например:

```text
Customer
 ├── Name (данные)
 ├── Address (данные)
 └── Drive() (действие)
```
