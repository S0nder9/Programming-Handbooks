## Свойства (Properties) и инкапсуляция

Свойство (**property**) — это специальный механизм в C#, который позволяет **безопасно получать и изменять поля класса**.

Обычно напрямую к `private` полям не обращаются — используют свойства.

---

## Поле класса

```csharp
private string _model = "";
```

- `private` означает, что поле доступно **только внутри класса** `Auto`.
- Это защищает данные от прямого изменения извне.

Пример (не сработает):

```csharp
Auto car = new Auto("Audi");
// car._model = "BMW"; // Ошибка
```

---

## Свойство `Model`

```csharp
public string Model
{
    get => _model;
    set => _model = value;
}
```

Разберём:

### `get`
Возвращает значение поля:

```csharp
string x = car.Model;
```

Это эквивалентно:

```csharp
return _model;
```

---

### `set`
Устанавливает новое значение:

```csharp
car.Model = "BMW";
```

`value` — это специальное ключевое слово, означающее переданное значение:

```csharp
_model = value;
```

То есть:

```text
value = "BMW"
```

---

## Полный пример

```csharp
using System;

namespace ConsoleApp2
{
    internal class Auto
    {
        private string _model = "";

        public string Model
        {
            get => _model;
            set => _model = value;
        }

        public Auto(string model)
        {
            Model = model;
            Console.WriteLine("An Obj of Car was created! " + model);
        }
    }
}
```

---

## Использование

```csharp
Auto audi = new Auto("Audi");

Console.WriteLine(audi.Model);

audi.Model = "BMW";

Console.WriteLine(audi.Model);
```

Результат:
```text
An Obj of Car was created! Audi
Audi
BMW
```

---

## Почему используем `Model = model`, а не `_model = model`?

Можно и так:

```csharp
_model = model;
```

Но лучше:

```csharp
Model = model;
```

Почему?
Потому что если позже добавить проверку:

```csharp
set
{
    if (value != "")
        _model = value;
}
```

она будет работать даже в конструкторе.

## Автосвойства (короткая запись)

Если дополнительная логика не нужна, можно короче:

```csharp
public string Model { get; set; }
```

C# сам создаст скрытое поле автоматически.

Это самый популярный вариант.
