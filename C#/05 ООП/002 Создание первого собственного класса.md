## Создание собственного класса (C#)

Класс — это шаблон для создания объектов.

Объект создаётся с помощью ключевого слова `new`.

---

## Класс `Auto`

```csharp
using System;

namespace ConsoleApp2
{
    internal class Auto
    {
        // Конструктор класса Auto, который инициализирует объект класса. У него нет параметров, и он не выполняет никаких действий при создании объекта.
        // Выполняется каждый раз, когда создается новый объект класса Auto, и может быть использован для установки начальных значений или выполнения других действий при создании объекта.
        public Auto()
        {
            Console.WriteLine("An object of Car was created!");
        }
    }
}
```

---

## Что такое конструктор?

**Конструктор** — это специальный метод, который:

- имеет то же имя, что и класс;
- не возвращает значение (даже `void` не пишется);
- вызывается автоматически при создании объекта.

```csharp
public Auto()
```

---

## Когда вызывается конструктор?

Каждый раз, когда создаётся объект:

```csharp
Auto audi = new Auto();
Auto bmw = new Auto();
```

---

## Что делает `new`?

```csharp
new Auto();
```

Оператор `new`:

1. Выделяет память под объект
2. Вызывает конструктор `Auto()`
3. Возвращает ссылку на объект

---

## Программа `Program`

```csharp
namespace ConsoleApp2
{
    internal class Program
    {
        static void Main(string[] args)
        {
            Auto audi = new Auto();
            Auto bmw = new Auto();

            Console.ReadKey();
        }
    }
}
```

---

## Что будет в выводе?

```text
An object of Car was created!
An object of Car was created!
```

---

## Важные моменты

- Каждый `new Auto()` вызывает конструктор отдельно
- Можно создавать сколько угодно объектов одного класса
- Класс — это “чертёж”, объект — “реальная вещь”

---

## Итог

```text
Class = описание
Object = экземпляр класса
Constructor = код, который запускается при создании объекта
```
