## Логика в `get` и использование нескольких свойств

В свойствах можно писать логику не только в `set`, но и в `get`.

Например, изменять возвращаемое значение в зависимости от других полей.

---

## Пример

```csharp
private bool _isLuxury;
private string _brand = "";

public string Brand
{
    get
    {
        if (_isLuxury)
        {
            return _brand + " - Luxury Edition";
        }
        else
        {
            return _brand;
        }
    }

    set => _brand = value;
}
```

---

## Как работает `get`

Если:

```csharp
_isLuxury = true;
_brand = "BMW";
```

то:

```csharp
Console.WriteLine(Brand);
```

Результат:

```text
BMW - Luxury Edition
```

---

Если:

```csharp
_isLuxury = false;
```

Результат:

```text
BMW
```

---

## Свойство `IsLuxury`

```csharp
public bool IsLuxury
{
    get => _isLuxury;
    set => _isLuxury = value;
}
```

Хранит:
- `true` — люксовая версия
- `false` — обычная

---

## Конструктор

Правильно:

```csharp
public Auto(string model, bool isLuxury, string brand)
{
    Model = model;
    Brand = brand;
    IsLuxury = isLuxury;

    Console.WriteLine("An Obj of Car was created! " + model);
}
```

---

## Использование

```csharp
Auto audi = new Auto("A8", true, "Audi");

Console.WriteLine(audi.Brand);
```

Результат:

```text
Audi - Luxury Edition
```

---

## Идея

`set` → контролирует **что записывать**  
`get` → контролирует **что возвращать**
