# Введение в классы и объекты

## Класс

Класс — это **шаблон (описание) объекта**.

Он определяет:
- свойства (данные, характеристики объекта);
- методы (поведение, действия объекта);
- может использоваться как **тип данных**;
- поддерживает **наследование** (создание новых классов на основе существующих).

---

## Объект

Объект — это **конкретный экземпляр класса**.

Он:
- создаётся по шаблону класса;
- имеет реальные значения свойств;
- использует методы класса для выполнения действий.
