## Простые массивы

```csharp
int[] myIntArr = new int[5]; // 5 целых чисел

myIntArr[0] = 5;
myIntArr[1] = 12;
myIntArr[2] = 12;
myIntArr[3] = 12;
myIntArr[4] = 12;
// myIntArr[5] = 12; // Ошибка Index was outside the bounds of the array.

Console.WriteLine(myIntArr[0]); // Доступ к 0 идексу в массиве: 5


Console.ReadKey();
```
