li/ul/ol/dl: Используются для создания списков. ul - маркированный список, ol - нумерованный список, li - элемент списка, а dl - список определений. div/section/aside/header/sidebar/footer: Элементы, используемые для структурирования веб-страницы. div - блочный элемент для группировки содержимого, section - для логического разделения разделов страницы, aside, header, sidebar и footer - для указания боковой панели, шапки, бокового блока и подвала соответственно. p: Определяет абзац текста. strong/em: Определяют важные или ударные части текста. strong для семантического выделения важности, em - для выделения эмфазы. img: Используется для вставки изображений. blockquote: Определяет цитаты, которые обычно выделены отступами или курсивом. head: Содержит метаданные документа, такие как заголовок страницы, ссылки на стили и скрипты. body: Содержит основное содержимое веб-страницы. meta: Используется для определения метаданных документа, таких как описание, ключевые слова и кодировка. title: Определяет заголовок документа, который отображается в строке заголовка браузера или на вкладке. src: Атрибут, используемый для указания источника ресурса, такого как изображение или видео. style: Используется для определения стилей непосредственно в HTML документе. link: Используется для связывания внешних ресурсов с документом, обычно используется для подключения стилей. noscript: Определяет содержимое, которое будет отображаться в случае, если браузер пользователя не поддерживает JavaScript или JavaScript отключен. form/input/textarea/button/select: Элементы формы для создания интерактивных элементов на странице, таких как поля ввода, кнопки и выпадающие списки. small/pre/span: Элементы для форматирования текста. small - для небольшого текста, pre - для сохранения форматирования текста, span - для выделения части текста. article/nav: article используется для независимых статей на странице, nav - для навигационных ссылок. table/thead/tbody/tfoot/tr/td: Элементы для создания таблиц. table - сама таблица, thead, tbody и tfoot - заголовок, тело и подвал таблицы соответственно, tr - строка таблицы, td - ячейка в строке. fieldset/legend: Используются для группировки связанных элементов формы. fieldset - для создания рамки вокруг группы элементов, legend - для создания заголовка для этой группы. label: Используется для создания метки для элемента формы. figure/figcaption: figure используется для встраивания медиаресурсов с подписью figcaption. canvas: Используется для рисования графики на веб-странице с помощью JavaScript. iframe: Используется для встраивания другого документа в текущий документ. embed: Используется для встраивания мультимедийных содержимых, таких как аудио или видео. hr/br: hr - горизонтальная линия, br - перевод строки. audio/video/source: Используются для встраивания аудио или видео на веб-страницу. source определяет источник мультимедийного контента. button: Элемент для создания кнопок. code/kbd: code используется для обозначения кода программы или компьютерного вывода, kbd - для обозначения клавиш на клавиатуре. figcaption: Используется для определения подписи к элементу figure. optgroup/option: optgroup используется для группировки элементов выпадающего списка option. progress: Используется для создания полосы прогресса. s/del: s и del используются для обозначения текста, который был удален.
